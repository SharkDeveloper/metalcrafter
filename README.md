# Shamonov.corp - Промышленная металлообработка

Профессиональная металлообработка и сварочные работы. Собственные цеха, станки ЧПУ, гарантия качества.

## Описание проекта

Этот проект представляет собой статический сайт для компании Shamonov.corp, специализирующейся на промышленной металлообработке. Сайт включает в себя несколько страниц с информацией об услугах, проектах, контактах и преимуществах компании.

## Запуск приложения локально

1. Убедитесь, что у вас установлен Python 3.7 или выше.

2. Создайте виртуальное окружение:
   ```bash
   python -m venv venv
   ```

3. Активируйте виртуальное окружение:
   - В Windows:
     ```bash
     venv\Scripts\activate
     ```
   - В macOS/Linux:
     ```bash
     source venv/bin/activate
     ```

4. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

5. Запустите приложение:
   ```bash
   python main.py
   ```

6. Откройте браузер и перейдите по адресу `http://localhost:8000`

## Деплой на Railway

Для автоматического деплоя на Railway:

1. Создайте аккаунт на [Railway](https://railway.app/)

2. Подключите ваш GitHub репозиторий к Railway

3. Railway автоматически обнаружит `railway.json` и настроит сборку и деплой

4. Приложение будет доступно по адресу, предоставленному Railway
